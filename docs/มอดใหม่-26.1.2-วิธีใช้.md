# คู่มือใช้งานมอดชุดใหม่ — Minecraft 26.1.2 (ลง 2026-06-28)

> อ้างอิงจากไฟล์ jar จริงใน `.minecraft\mods\` ทุกตัว — ชื่อไอเทม/สูตร/ปุ่ม/ดรอป ดึงจากในมอดตรง ๆ
> ค่าที่ตั้งได้ในเกม (เช่น ปุ่มลัด, ค่าใน config) จะวงเล็บไว้ว่า "ตั้งเอง" — ของพวกนี้เปิดดู/แก้ได้ใน **Mod Menu → ปุ่มเฟือง** ของแต่ละมอด

---

## TL;DR — 3 มอดที่เปลี่ยนวิธีเล่นมากสุด

1. **Waystones** — วางเสา "Waystone" คลิกขวาตั้งชื่อ = จุดวาร์ป เดินทางข้ามแมพฟรี/เสียค่า XP เล็กน้อย แชร์กับเพื่อนได้ผ่าน Sharestone
2. **Traveler's Backpack** — เป้สวมหลังจุของเยอะมาก + อัปเกรดเตา/โต๊ะคราฟต์/แม่เหล็กดูดของ/ถังน้ำ + ถุงนอนในตัว กางนอนได้ทุกที่
3. **VeinMiner** — กดปุ่มค้างแล้วขุดแร่ทีเดียวทั้งสาย (และตัดไม้/ขุดดินทั้งกอง) เลือกรูปทรงขุดได้ (ปกติ/อุโมงค์/แนวราบ)

> เสริม: **Combatify** รื้อระบบตีใหม่ (อิงจาก Combat Test ของ Mojang) + เพิ่มมีดสั้น/ดาบยาว/โล่ตามวัสดุ — **MutantMonsters** ใส่มินิบอส 5 ตัวที่ดรอปของแรง

---

## ตารางภาพรวมทั้งชุด

| มอด | ทำอะไร | ใช้ยังไง (โดยย่อ) | ฝั่ง |
|---|---|---|---|
| **VeinMiner** | ขุดแร่/ตัดไม้ทีเดียวทั้งสาย | กดปุ่มค้าง (ตั้งเอง) ขณะขุด | client+server |
| **Waystones** | เสาวาร์ปข้ามแมพ + แชร์ทีม | คราฟต์ Waystone → คลิกขวาตั้งชื่อ | client+server |
| **Traveler's Backpack** | เป้จุของ + อัปเกรด + ถุงนอน | คราฟต์ → คลิกขวาที่มือสวมหลัง → กดปุ่มเปิด | client+server |
| **Combatify** | รื้อระบบคอมแบท + อาวุธ/โล่ใหม่ | ติดตั้งแล้วทำงานเลย; อาวุธใหม่ต้องเปิด pack เสริม | client+server |
| **MutantMonsters** | มินิบอส 5 ตัว + ของดรอปแรง | เจอแบบสุ่มที่ม็อบปกติเกิด | client+server |
| **Advanced Netherite** | เกราะ/อาวุธสายเหนือ netherite | คราฟต์ต่อยอดจาก netherite | client+server |
| **Dungeons & Taverns** | ดันเจี้ยน/โครงสร้างใหม่ใน worldgen | เจอตอนสำรวจ (เฉพาะแมพที่เจนใหม่) | server |
| **Easy Anvils** | ทั่งไม่พัง + ไม่มี "Too Expensive" | ใช้ทั่งปกติ ตีได้ไม่จำกัด | client+server |
| **Comforts** | ถุงนอน/เปลญวน นอนได้ทุกที่ | คราฟต์ → คลิกขวาวางแล้วนอน | client+server |
| **Highlighter** | ไฮไลต์ของที่เพิ่งเก็บใน inventory | อัตโนมัติ ไม่ต้องกดอะไร | client only |

## เจาะรายตัว — วิธีใช้จริงในเกม

### ⛏️ VeinMiner — ขุดทั้งสายทีเดียว
ขุดบล็อกเดียวแล้วบล็อกชนิดเดียวกันที่ติดกันทั้งก้อนแตกตามทันที (แร่, ไม้, ดิน, หิน ฯลฯ)

**วิธีใช้:**
1. ไป **Options → Controls → หมวด "Veinminer"** ผูกปุ่มก่อน (มาตรฐานยังไม่ผูกให้)
   - **Veinmining (Hold)** — กดค้างขณะขุด = ขุดทั้งสาย (แนะนำ ปลอดภัยสุด)
   - **Veinmining (Toggle)** — กดสลับเปิด/ปิดถาวร
   - **Open Client Settings** — เปิดหน้าตั้งค่า client
2. เล็งแร่ → กดปุ่มที่ผูกไว้ค้าง → ขุด → ทั้งสายหายรวด

**รูปทรงการขุด (เปลี่ยนในหน้า Client Settings):**

| รูปทรง | ผล |
|---|---|
| Normal | ตามสายบล็อกชนิดเดียวกันที่ติดกัน (ดีฟอลต์) |
| Tunnel 1x1 / 2x2 / 3x3 | เจาะเป็นอุโมงค์ตามขนาด |
| Flat | ขุดเป็นแผ่นแนวราบ |

> ⚠️ กินความทนทานจอบ/อีเต้อตามจำนวนบล็อกที่แตก — สายยาว ๆ พลั่วหมดสภาพไว พกของซ่อม/Mending
> 💡 ตัว `veinminer-client` (มีในเครื่อง) คือส่วนที่ให้ปุ่ม+รูปทรง ถ้าเล่นเซิร์ฟเพื่อน เซิร์ฟมี veinminer หลักอยู่แล้วใช้ได้เลย

### 🪧 Waystones — เสาวาร์ปข้ามแมพ
หัวใจคือ **Warp Stone** (วัตถุดิบกลาง) เอาไปคราฟต์ทุกอย่างต่อ

**สูตรหลัก:**

| ไอเทม | สูตร | ได้ |
|---|---|---|
| **Warp Stone** | amethyst×4 + ender pearl×4 + gold ingot×1 (AEA/EGE/AEA) | 1 (ของวาร์ปพกติดตัว มีคูลดาวน์) |
| **Waystone** (เสาหลัก) | หินตกแต่ง×6 + Warp Stone×1 (` S `/`SWS`/`SSS`) | 1 เสา |
| **Warp Scroll** | gold nugget×4 + ink sac×1 + amethyst×1 + paper×3 | 3 (ม้วนวาร์ปใช้ครั้งเดียว ไปได้ทุกเสาที่เปิดแล้ว) |
| **Return Scroll** | gold nugget×4 + ink sac×1 + paper×3 | 3 (กลับเสาที่ผูก/ใกล้สุด ใช้ครั้งเดียว) |
| **Sharestone** (แชร์ทีม) | stone bricks×6 + copper ingot×2 + Warp Stone×1 | 1 (เสาแชร์ — ใครอยู่เครือข่ายสีเดียวกันวาร์ปหากันได้) |
| **Warp Plate** | stone bricks×4 + amethyst×4 + flint×1 | 1 (แผ่นเหยียบวาร์ปอัตโนมัติ) |

> 🪨 **"หินตกแต่ง" ของ Waystone เลือกได้หลายแบบ** (จากในมอด): deepslate, chiseled sandstone, andesite, blackstone, prismarine, purpur, end stone, mud bricks, red nether bricks — แบบไหนก็ได้ ×6 + Warp Stone×1 วางทรง ` S `/`SWS`/`SSS` เหมือนกันหมด · **Sharestone** ต่างหากที่ใช้ stone bricks×6

**วิธีผูกจุด:**
1. วาง Waystone กับพื้น → **คลิกขวา** → ใส่ชื่อ (เช่น "บ้าน", "เหมือง") → กดยืนยัน = เสานี้ "เปิด" แล้ว
2. ไปวางเสาที่อื่นอีก ทำซ้ำ
3. **คลิกขวาที่เสาใดก็ได้** → ขึ้นเมนูรายชื่อเสาที่เปิดแล้ว → คลิกชื่อ = วาร์ปไป
4. วาร์ปจากที่ไหนก็ได้ด้วย **Warp Stone** (คลิกขวาค้าง เลือกเสา) — มีคูลดาวน์

**ลูกเล่นในเมนูเสา (ปุ่ม Manage / Personal Settings):**
- จัดเรียงเสา (ลากเรียง, double-click ดันขึ้นบนสุด), ตั้ง Favorite, จัดกลุ่ม (Group)
- เรียงตาม: Manual / ชื่อ / ระยะทาง
- ตั้งการมองเห็นเสา: เปิดเมื่อ activate / เห็นทุกคน (Global) / ผ่าน Attuned Shard เท่านั้น

> 💰 การวาร์ปอาจ **เสีย XP** ตามระยะ (ตั้งได้ใน config — ปิดให้ฟรีก็ได้) เมนูจะบอก "Level Cost / Experience Cost" ก่อนวาร์ป
> 👥 เล่นกับเพื่อน: ใช้ **Sharestone** ทำเครือข่ายร่วม หรือเปิดเสาเป็น **Global** ให้ทุกคนเห็น

### 🎒 Traveler's Backpack — เป้สวมหลังสารพัดประโยชน์
จุของเยอะ ใส่อัปเกรดได้ มีถุงนอนในตัว

**ลำดับคราฟต์ (ทำตามนี้):**
1. **ถุงนอน TB** = ขนแกะ×3 เรียงแนวนอน (`##X`) — *จำเป็นต่อสูตรเป้* (advancement "Getting Started")
2. **Backpack Tank** (ออปชัน) = แก้วใส×6 + iron ingot×2 (`BAB`/`B B`/`BAB`)
3. **เป้ Standard** = หนัง×4 + เชือก×1 + หีบไม้×1 + Backpack Tank×2 + ถุงนอน TB×1
   - เรียง: `XBX` / `CDC` / `XSX` (X=หนัง, B=เชือก, C=tank, D=หีบ, S=ถุงนอน)

**สวม & เปิด:**
- **สวม:** ถือเป้ไว้มือหลัก → คลิกขวา (หรือเปิดช่อง Back Slot Integration เอาใส่ช่องหลังตรง ๆ)
- **เปิด:** กดปุ่ม **Open Backpack** (ตั้งเองใน Controls หมวด "Traveler's Backpack")
- ปุ่มอื่นในหมวดเดียวกัน: Sort Backpack, **Enable/Disable Special Ability**, **Swap Tool/Change Hose Mode** (ดีฟอลต์ Z), Toggle Upgrade ช่อง 1–4

**ยกระดับ Tier (จุเพิ่ม):** ใช้ **Smithing Table** — เอา **Leather เป็น template** + เป้ tier ปัจจุบัน → เลื่อนขั้น Leather → Iron → Gold → Diamond → Netherite

**อัปเกรด** (วางในช่องด้านขวาของหน้าเป้):

| อัปเกรด | ทำอะไร |
|---|---|
| Crafting | โต๊ะคราฟต์ 3×3 ในเป้ |
| Furnace / Smoker / Blast Furnace | เตาเผา/รมควัน/หลอมในเป้ |
| Pickup | ของที่เก็บได้ไหลเข้าเป้อัตโนมัติ |
| Magnet | ดูดของรอบตัวตามรัศมี (ตั้งได้) |
| Feeding | กินอาหารจากเป้ให้อัตโนมัติ |
| Refill | เติมไอเทมที่เลือกกลับเข้า inventory อัตโนมัติ |
| Void | ถังขยะ + ฟิลเตอร์ลบของที่ไม่อยากได้ |
| Tanks | เพิ่มถังของเหลว 2 ถัง |
| Lantern / Jukebox | ไฟส่องสว่าง / เปิดเพลง |

**ถุงนอนในเป้:** เปิดหน้าเป้ → ปุ่ม **Use Sleeping Bag** (ถ้าไม่เห็น กดปุ่ม 3 จุด) นอนได้เลยไม่ต้องพกเตียง
**ความสามารถพิเศษ (Ability):** เป้แต่ละวัสดุมีพลังต่างกัน เช่น Cactus เติมน้ำตอนฝนตก, Creeper ระเบิดกันตาย, Cake ป้อนอาหาร+ฟื้นเลือด, Spider ปีนกำแพง — กดปุ่ม Ability เปิด/ปิด

> 💧 **Hose (สายยาง):** สวมเป้แล้วถือ Hose จิ้มของเหลวในถัง — กด Z สลับโหมด/ถัง เติม/เทน้ำ-ลาวา-โพชั่นได้
> ☠️ ของในเป้ตอนตาย: ตั้งได้ใน config (Backpack Death Place / Void Protection) ว่าจะตกพื้นหรือกันหาย

### ⚔️ Combatify — รื้อระบบคอมแบทใหม่
อิงจาก **Combat Test snapshots** ของ Mojang (ระบบตีทดลองที่ไม่เคยลงเกมจริง) — เปลี่ยน "ฟีล" การต่อสู้ทันทีที่ลง ไม่ต้องคราฟต์อะไร

**สิ่งที่เปลี่ยน (รวม ๆ):**
- ระยะเอื้อมโจมตี (attack reach) และจังหวะคูลดาวน์ตีต่างจาก vanilla
- **โล่** ทำงานละเอียดขึ้น (มีโล่หลายระดับวัสดุ, กลไกบล็อก/โดนปลดต่างเดิม)
- เพิ่ม **หมวดอาวุธ**: knife (มีดสั้น), longsword (ดาบยาว), spear, mace ฯลฯ — ของเดิมถูกจัดหมวดใหม่

**อาวุธ/โล่ใหม่ที่คราฟต์ได้** (ทุกวัสดุ: wood/stone/copper/iron/gold/diamond/netherite):
- **Knife** = วัสดุ×1 วางบน stick×1 (แนวตั้ง `X`/`#`) → ตีเร็ว ดาเมจน้อย
- **Longsword** = วัสดุ×2 + Knife วัสดุเดียวกัน×1 (`X`/`X`/`#` โดย #=knife) → ระยะเอื้อมไกล
- **Shield** ตามวัสดุ (copper/iron/gold/diamond/netherite) + ย้อมสีได้ 16 สี

> ⚠️ **อาวุธใหม่ (knife/longsword/โล่ตามวัสดุ) อยู่ใน pack เสริมในตัวมอด** — ต้องเปิด built-in datapack/resourcepack ของ Combatify ก่อนถึงจะคราฟต์ได้ (เปิดใน **Mod Menu → Combatify** หรือเมนู Datapacks ตอนสร้างโลก) ⚠️ ตัวเลขดาเมจ/ระยะที่แน่นอน ต้องเช็คในเกม
> 🛡️ Combatify มี option ปรับเยอะ (สไตล์ vanilla/combat-test, กลไกโล่หลายแบบ) — เข้าไปจูนใน Mod Menu ได้ ตั้งให้ตรงกับเซิร์ฟเพื่อน

### 👹 MutantMonsters — มินิบอส 5 ตัว
ม็อบกลายพันธุ์ตัวใหญ่ เลือดเยอะ ตีแรง **เกิดแบบสุ่ม (โอกาสน้อย) ตรงที่ม็อบปกติชนิดนั้นเกิด** — เจอแล้วเตรียมรับมือ

| มินิบอส | เจอจาก | ดรอป/ของเด่น |
|---|---|---|
| **Mutant Zombie** | โซมบี้ (สุ่ม) | **Hulk Hammer** (ค้อนทุบแรง) — ล้มแล้วยังฟื้น ต้องจุดไฟด้วย flint & steel ตอนมันล้มถึงตายสนิท |
| **Mutant Skeleton** | สเกเลตัน | ชิ้นส่วนกระดูก (limb/rib/pelvis/shoulder pad) + **Skull** → คราฟต์เป็น **ชุดเกราะ Mutant Skeleton** |
| **Mutant Creeper** | ครีปเปอร์ | **Creeper Shard** (ตัว charged ดรอปมากกว่า) ใช้ทำ Creeper Minion |
| **Mutant Enderman** | เอนเดอร์แมน | **Endersoul Hand** (ไอเทมเทเลพอร์ตตัวเอง) + Endersoul Fragment |
| **Spider Pig** | สร้างเอง | ป้อนหมู fermented spider eye แล้วปา **Chemical X** ใส่ |

**ชุดเกราะ Mutant Skeleton (คราฟต์จากชิ้นส่วน) — โบนัสติดเกราะ:**
- Boots = กระโดดสูงขึ้น | Leggings = วิ่งเร็วขึ้น | Chestplate = ขึ้นธนูไว (quick draw) | Skull/หัว = ปลดล็อก multishot ให้ธนู

**ของพิเศษอื่น:**
- **Chemical X** — ได้เป็นโพชั่น ใช้สร้าง Mutant Snow Golem / Spider Pig
- **Creeper Minion** — ฟักจาก Creeper Minion Egg เป็นสัตว์เลี้ยง ใช้ **Creeper Minion Tracker** ตั้งค่ารัศมีระเบิด/ขี่บ่า/โชว์ชื่อ
- **Hulk Hammer** — ใช้ฆ่า Mutant Zombie ได้ปั๊บ (advancement "Hulk Smash!")

> ⚠️ Mutant Zombie ล้มแล้ว "ไม่ตายจริง" จนกว่าจะจุดไฟทับ — อย่าเพิ่งวางใจ
> 💀 ตัวพวกนี้แรงพอจะฆ่าคนเกราะเหล็กได้ ควรมีโล่ (Combatify) + เกราะดี + อาหารก่อนสู้

### 🔨 Easy Anvils — ทั่งไม่พัง ตีได้ไม่จำกัด
ใช้ทั่ง vanilla ตามปกติ แต่:
- **ทั่งไม่เสื่อม/ไม่แตก** (ไม่มี chipped/damaged anvil) — ทุบใช้ได้ตลอด
- **ตัด "Too Expensive!"** ทิ้ง — รวมเอนแชนต์/ซ่อมได้เรื่อย ๆ ไม่ตันที่ 39 เลเวล
- **เปลี่ยนชื่อใส่สี/ฟอร์แมตได้** — ใช้รหัส `§` (สี, ตัวหนา, เอียง ฯลฯ) ในช่องตั้งชื่อทั่ง

> 💡 คู่กับ Mending: เอาของหลายชิ้นมารวมเอนแชนต์ในทั่งได้ถูกลงเยอะเมื่อไม่มีเพดาน Too Expensive

### 🛏️ Comforts — ถุงนอน & เปลญวน นอนได้ทุกที่
นอนข้ามคืน/ข้ามวันโดย **ไม่ต้องตั้ง spawn** และกัน Phantom ได้ (ตั้งใน config)

| ไอเทม | สูตร | ใช้ยังไง |
|---|---|---|
| **Sleeping Bag** | ขนแกะ×3 เรียงแนวตั้ง (` # `×3) | ย่อตัว (crouch) แล้วคลิกขวาวางพื้น → นอน; **มีโอกาสขาดหลังใช้** |
| **Hammock** (เปลญวน) | ขนแกะ×1 + เชือก×2 + แท่งไม้×2 (` X `/`S#S`/` X `) | ต้องแขวนระหว่าง **Rope and Nail** สองอัน; นอนได้ตอน **กลางวัน** (ข้ามไปกลางคืน) |
| **Rope and Nail** | เชือก×4 + iron ingot×1 (`SS`/`SI`/`S`) | ได้ 2 อัน — สำหรับแขวนเปลญวน |

**ขั้นตอนเปลญวน:** วาง Rope and Nail สองอันห่างกันพอดี → คลิกขวาวาง Hammock เชื่อมระหว่างสองอัน → คลิกขวานอน (กลางวันเท่านั้น)

> 💤 Sleeping Bag = นอนเร็วทุกที่แต่เปลือง (ขาดได้), Hammock = ถาวรกว่าแต่ต้องตั้งเสา + นอนกลางวัน
> 👻 ทั้งคู่ตั้งให้ "พยายามนอน = หยุด Phantom เกิด" ได้แม้ไม่ได้นอนจริง (config: hammocksStopPhantoms / sleepingBagsStopPhantoms)

### ✨ Highlighter — ไฮไลต์ของที่เพิ่งเก็บ
**Client only — ทำงานอัตโนมัติ ไม่มีปุ่ม** ของชิ้นที่เพิ่งเก็บเข้า inventory จะมี **ขอบเรืองไฮไลต์** ในหน้า inventory ให้เห็นว่าได้อะไรมาใหม่ (กันพลาด/กันหาของ)

> หมายเหตุ: เป็นไฮไลต์ "ของในช่อง inventory" ตอนเปิดกระเป๋า ไม่ใช่ไฮไลต์ไอเทมที่ตกอยู่บนพื้นในโลก

## 🔗 คอมโบเด็ด — ใช้คู่กันดี

1. **VeinMiner + เป้ (Pickup/Magnet upgrade)** → ขุดแร่ทั้งสายทีเดียว ของเด้งเข้าเป้เองหมด ไม่ต้องไล่เก็บ
2. **Waystones + เป้** → ตั้งเสาที่เหมือง/ฟาร์ม/บ้าน ขนของเต็มเป้แล้ววาร์ปกลับ ลดเวลาเดินทาง
3. **Comforts/ถุงนอนในเป้ + Waystones** → ออกสำรวจไกล นอนข้ามคืนทุกที่ + กัน Phantom + วาร์ปกลับเมื่อเสร็จ
4. **Combatify (โล่หลายระดับ) + MutantMonsters** → เจอมินิบอส มีโล่ดี ๆ บล็อกไหว รอดสูงขึ้น
5. **Easy Anvils + ของดรอปบอส/Advanced Netherite** → รวมเอนแชนต์เกราะ-อาวุธชุดลุยบอสได้ถูก ไม่ตัน Too Expensive
6. **เกราะ Mutant Skeleton + ธนู** → quick draw + multishot ยิงรัวหลายดอก เปิดศึกระยะไกลกับมินิบอสตัวอื่น
