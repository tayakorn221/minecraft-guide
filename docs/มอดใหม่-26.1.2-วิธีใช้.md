# คู่มือใช้งานมอดชุดใหม่ — Minecraft 26.1.2 (ลง 2026-06-28)

> ข้อมูลทุกอย่างดึงตรงจากไฟล์ jar จริงใน `.minecraft\mods\` — ชื่อไอเทม สูตร ปุ่ม ดรอป เอามาจากในมอดเป๊ะ ๆ
> ค่าไหนที่ตั้งได้เองในเกม (เช่น ปุ่มลัด หรือค่าใน config) จะวงเล็บกำกับว่า "ตั้งเอง" ไว้ — ของพวกนี้เปิดดู/แก้ได้ใน **Mod Menu → ปุ่มเฟือง** ของแต่ละมอด

---

## TL;DR — 3 มอดที่เปลี่ยนวิธีเล่นมากสุด

1. **Waystones** — วางเสา "Waystone" คลิกขวาตั้งชื่อ ก็ได้จุดวาร์ปแล้ว เดินทางข้ามแมพได้ฟรี (หรือเสียค่า XP นิดหน่อย) แถมแชร์กับเพื่อนได้ผ่าน Sharestone
2. **Traveler's Backpack** — เป้สวมหลังที่จุของเยอะโคตร ใส่อัปเกรดได้ทั้งเตา/โต๊ะคราฟต์/แม่เหล็กดูดของ/ถังน้ำ แถมมีถุงนอนในตัว กางนอนได้ทุกที่
3. **VeinMiner** — กดปุ่มค้างแล้วขุดแร่ทีเดียวหมดทั้งสาย (ตัดไม้/ขุดดินทั้งกองก็ได้) แถมเลือกรูปทรงขุดได้ด้วย (ปกติ/อุโมงค์/แนวราบ)

> เสริม: **Combatify** รื้อระบบตีใหม่ทั้งยวง (อิงจาก Combat Test ของ Mojang) แล้วเพิ่มมีดสั้น/ดาบยาว/โล่ตามวัสดุเข้ามา — ส่วน **MutantMonsters** ใส่มินิบอส 5 ตัวที่ดรอปของแรง ๆ

---

## ตารางภาพรวมทั้งชุด

| มอด | ทำอะไร | ใช้ยังไง (โดยย่อ) | ฝั่ง |
|---|---|---|---|
| **VeinMiner** | ขุดแร่/ตัดไม้ทีเดียวทั้งสาย | กดปุ่มค้าง (ตั้งเอง) ตอนขุด | client+server |
| **Waystones** | เสาวาร์ปข้ามแมพ + แชร์ทีม | คราฟต์ Waystone → คลิกขวาตั้งชื่อ | client+server |
| **Traveler's Backpack** | เป้จุของ + อัปเกรด + ถุงนอน | คราฟต์ → ถือไว้คลิกขวาสวมหลัง → กดปุ่มเปิด | client+server |
| **Combatify** | รื้อระบบคอมแบท + อาวุธ/โล่ใหม่ | ลงแล้วทำงานเลย ส่วนอาวุธใหม่ต้องเปิด pack เสริมก่อน | client+server |
| **MutantMonsters** | มินิบอส 5 ตัว + ของดรอปแรง | โผล่แบบสุ่มตรงที่ม็อบปกติเกิด | client+server |
| **Advanced Netherite** | เกราะ/อาวุธเหนือ netherite — 4 สาย (netherite+diamond/emerald/gold/iron) ครบชุด + มี spear | คราฟต์ต่อยอดจาก netherite ingot | client+server |
| **Dungeons & Taverns** | ดันเจี้ยน/โครงสร้างใหม่ใน worldgen | เจอตอนสำรวจ (เฉพาะแมพที่เจนใหม่) | server |
| **Easy Anvils** | ทั่งไม่พัง + ไม่มี "Too Expensive" | ใช้ทั่งปกติ ตีได้ไม่จำกัด | client+server |
| **Comforts** | ถุงนอน/เปลญวน นอนได้ทุกที่ | คราฟต์ → คลิกขวาวางแล้วนอน | client+server |
| **Highlighter** | ไฮไลต์ของที่เพิ่งเก็บใน inventory | ทำงานอัตโนมัติ ไม่ต้องกดอะไร | client only |

## เจาะรายตัว — วิธีใช้จริงในเกม

### ⛏️ VeinMiner — ขุดทั้งสายทีเดียว
ขุดบล็อกเดียว แล้วบล็อกชนิดเดียวกันที่ติดกันทั้งก้อนจะแตกตามทันที (แร่, ไม้, ดิน, หิน ฯลฯ)

**วิธีใช้:**
1. ไป **Options → Controls → หมวด "Veinminer"** ผูกปุ่มก่อน (เริ่มมายังไม่ผูกให้)
   - **Veinmining (Hold)** — กดค้างตอนขุด = ขุดทั้งสาย (แนะนำตัวนี้ ปลอดภัยสุด)
   - **Veinmining (Toggle)** — กดสลับเปิด/ปิดค้างไว้
   - **Open Client Settings** — เปิดหน้าตั้งค่าฝั่ง client
2. เล็งแร่ → กดปุ่มที่ผูกไว้ค้าง → ขุด → ทั้งสายหายรวดเดียว

**รูปทรงการขุด (เปลี่ยนในหน้า Client Settings):**

| รูปทรง | ผล |
|---|---|
| Normal | ตามสายบล็อกชนิดเดียวกันที่ติดกัน (ดีฟอลต์) |
| Tunnel 1x1 / 2x2 / 3x3 | เจาะเป็นอุโมงค์ตามขนาด |
| Flat | ขุดเป็นแผ่นแนวราบ |

> ⚠️ กินความทนทานของจอบ/อีเต้อตามจำนวนบล็อกที่แตก — สายยาว ๆ พลั่วหมดสภาพไว พกของซ่อมหรือติด Mending ไว้
> 💡 ตัว `veinminer-client` (ที่มีในเครื่อง) คือส่วนที่ให้ปุ่มกับรูปทรงขุด ถ้าเล่นเซิร์ฟเพื่อน เซิร์ฟมี veinminer ตัวหลักอยู่แล้วก็ใช้ได้เลย

### 🪧 Waystones — เสาวาร์ปข้ามแมพ
หัวใจของมอดนี้คือ **Warp Stone** (วัตถุดิบกลาง) เอาไปต่อยอดคราฟต์ของอื่นได้หมด

**สูตรหลัก:**

| ไอเทม | สูตร | ได้ |
|---|---|---|
| **Warp Stone** | amethyst×4 + ender pearl×4 + gold ingot×1 (AEA/EGE/AEA) | 1 (ของวาร์ปพกติดตัว มีคูลดาวน์) |
| **Waystone** (เสาหลัก) | หินตกแต่ง×6 + Warp Stone×1 (` S `/`SWS`/`SSS`) | 1 เสา |
| **Warp Scroll** | gold nugget×4 + ink sac×1 + amethyst×1 + paper×3 | 3 (ม้วนวาร์ปใช้ครั้งเดียว ไปได้ทุกเสาที่เปิดแล้ว) |
| **Return Scroll** | gold nugget×4 + ink sac×1 + paper×3 | 3 (กลับเสาที่ผูก/ใกล้สุด ใช้ครั้งเดียว) |
| **Sharestone** (แชร์ทีม) | stone bricks×6 + copper ingot×2 + Warp Stone×1 | 1 (เสาแชร์ — ใครอยู่เครือข่ายสีเดียวกันวาร์ปหากันได้) |
| **Warp Plate** | stone bricks×4 + amethyst×4 + flint×1 | 1 (แผ่นเหยียบวาร์ปอัตโนมัติ) |

> 🪨 **"หินตกแต่ง" ของ Waystone เลือกได้หลายแบบ** (จากในมอด): deepslate, chiseled sandstone, andesite, blackstone, prismarine, purpur, end stone, mud bricks, red nether bricks — แบบไหนก็ได้ ×6 + Warp Stone×1 วางทรง ` S `/`SWS`/`SSS` เหมือนกันหมด · **Sharestone** ต่างหากที่ใช้ stone bricks×6

**วิธีผูกจุด:**
1. วาง Waystone ลงพื้น → **คลิกขวา** → ใส่ชื่อ (เช่น "บ้าน", "เหมือง") → กดยืนยัน เสานี้ก็ "เปิด" แล้ว
2. ไปวางเสาที่อื่นต่อ ทำซ้ำแบบเดิม
3. **คลิกขวาที่เสาไหนก็ได้** → จะขึ้นเมนูรายชื่อเสาที่เปิดไว้ → คลิกชื่อก็วาร์ปไปเลย
4. อยากวาร์ปจากที่ไหนก็ได้ ใช้ **Warp Stone** (คลิกขวาค้างแล้วเลือกเสา) — แต่มีคูลดาวน์นะ

**ลูกเล่นในเมนูเสา (ปุ่ม Manage / Personal Settings):**
- จัดเรียงเสา (ลากเรียงเอง หรือ double-click ดันขึ้นบนสุด), ปักเป็น Favorite, จัดกลุ่ม (Group)
- เรียงตาม: Manual / ชื่อ / ระยะทาง
- ตั้งการมองเห็นเสาได้: เปิดเมื่อ activate / เห็นกันหมดทุกคน (Global) / ต้องมี Attuned Shard ถึงเห็น

> 💰 การวาร์ปอาจ **เสีย XP** ตามระยะทาง (ตั้งได้ใน config จะปิดให้ฟรีก็ได้) เมนูจะบอก "Level Cost / Experience Cost" ให้เห็นก่อนวาร์ป
> 👥 เล่นกับเพื่อน: ใช้ **Sharestone** ทำเครือข่ายร่วมกัน หรือเปิดเสาเป็น **Global** ให้ทุกคนเห็นก็ได้

### 🎒 Traveler's Backpack — เป้สวมหลังสารพัดประโยชน์
จุของเยอะ ใส่อัปเกรดได้สารพัด แถมมีถุงนอนในตัว

**ลำดับคราฟต์ (ทำตามนี้เลย):**
1. **ถุงนอน TB** = ขนแกะ×3 เรียงแนวนอน (`##X`) — *จำเป็นต่อสูตรเป้* (advancement "Getting Started")
2. **Backpack Tank** (ออปชัน) = แก้วใส×6 + iron ingot×2 (`BAB`/`B B`/`BAB`)
3. **เป้ Standard** = หนัง×4 + เชือก×1 + หีบไม้×1 + Backpack Tank×2 + ถุงนอน TB×1
   - เรียง: `XBX` / `CDC` / `XSX` (X=หนัง, B=เชือก, C=tank, D=หีบ, S=ถุงนอน)

**สวม & เปิด:**
- **สวม:** ถือเป้ไว้มือหลัก → คลิกขวา (หรือจะเปิดช่อง Back Slot Integration แล้วใส่ช่องหลังตรง ๆ ก็ได้)
- **เปิด:** กดปุ่ม **Open Backpack** (ตั้งเองใน Controls หมวด "Traveler's Backpack")
- ปุ่มอื่นในหมวดเดียวกัน: Sort Backpack, **Enable/Disable Special Ability**, **Swap Tool/Change Hose Mode** (ดีฟอลต์ปุ่ม Z), Toggle Upgrade ช่อง 1–4

**ยกระดับ Tier (เพิ่มความจุ):** ใช้ **Smithing Table** — เอา **Leather เป็น template** วางคู่กับเป้ tier ปัจจุบัน → ไล่ขั้น Leather → Iron → Gold → Diamond → Netherite

**อัปเกรด** (วางในช่องด้านขวาของหน้าเป้):

| อัปเกรด | ทำอะไร |
|---|---|
| Crafting | โต๊ะคราฟต์ 3×3 ในเป้ |
| Furnace / Smoker / Blast Furnace | เตาเผา/รมควัน/หลอมในเป้ |
| Pickup | ของที่เก็บได้ไหลเข้าเป้ให้อัตโนมัติ |
| Magnet | ดูดของรอบตัวตามรัศมี (ตั้งได้) |
| Feeding | ป้อนอาหารจากเป้ให้อัตโนมัติ |
| Refill | เติมไอเทมที่เลือกไว้กลับเข้า inventory ให้อัตโนมัติ |
| Void | ถังขยะ + ฟิลเตอร์คอยลบของที่ไม่อยากได้ |
| Tanks | เพิ่มถังของเหลวอีก 2 ถัง |
| Lantern / Jukebox | ไฟส่องสว่าง / เปิดเพลง |

**ถุงนอนในเป้:** เปิดหน้าเป้ → กดปุ่ม **Use Sleeping Bag** (ถ้าไม่เห็น กดปุ่ม 3 จุด) นอนได้เลยโดยไม่ต้องพกเตียง
**ความสามารถพิเศษ (Ability):** เป้แต่ละวัสดุมีพลังไม่เหมือนกัน เช่น Cactus เติมน้ำตอนฝนตก, Creeper ระเบิดกันตาย, Cake ป้อนอาหาร+ฟื้นเลือด, Spider ปีนกำแพงได้ — กดปุ่ม Ability เพื่อเปิด/ปิด

> 💧 **Hose (สายยาง):** สวมเป้แล้วถือ Hose จิ้มของเหลวในถังได้เลย — กด Z สลับโหมด/สลับถัง เติมหรือเทน้ำ-ลาวา-โพชั่นได้หมด
> ☠️ ของในเป้ตอนตาย: ตั้งได้ใน config (Backpack Death Place / Void Protection) ว่าจะให้ตกพื้นหรือกันหายไว้

### ⚔️ Combatify — รื้อระบบคอมแบทใหม่
อิงจาก **Combat Test snapshots** ของ Mojang (ระบบตีทดลองที่ไม่เคยลงเกมจริงสักที) — เปลี่ยน "ฟีล" การต่อสู้ทันทีที่ลง ไม่ต้องคราฟต์อะไรเลย

**สิ่งที่เปลี่ยนไป (รวม ๆ):**
- ระยะเอื้อมโจมตี (attack reach) กับจังหวะคูลดาวน์ตี ต่างจาก vanilla
- **โล่** ทำงานละเอียดขึ้น (มีโล่หลายระดับตามวัสดุ, กลไกบล็อก/โดนปลดก็ต่างจากเดิม)
- เพิ่ม **อาวุธใหม่ครบทุกวัสดุ** (wood→netherite): **knife** (มีดสั้น), **longsword** (ดาบยาว), **shield** (โล่โลหะ) — ⚠️ **ไม่มี spear/mace** (spear ไปอยู่ใน Advanced Netherite ต่างหาก)

**อาวุธ/โล่ใหม่ที่คราฟต์ได้** (ทุกวัสดุ: wood/stone/copper/iron/gold/diamond/netherite):
- **Knife** = วัสดุ×1 วางบน stick×1 (แนวตั้ง `X`/`#`) → ตีเร็ว แต่ดาเมจน้อย
- **Longsword** = วัสดุ×2 + Knife วัสดุเดียวกัน×1 (`X`/`X`/`#` โดย #=knife) → แลกมาด้วยระยะเอื้อมที่ไกลขึ้น
- **Shield** ทำตามวัสดุได้ (copper/iron/gold/diamond/netherite) แถมย้อมสีได้ 16 สี

> ⚠️ **อาวุธใหม่ (knife/longsword/โล่ตามวัสดุ) อยู่ใน pack เสริมที่มากับตัวมอด** — ต้องเปิด built-in datapack/resourcepack ของ Combatify ก่อนถึงจะคราฟต์ได้ (เปิดใน **Mod Menu → Combatify** หรือเมนู Datapacks ตอนสร้างโลก) ⚠️ ตัวเลขดาเมจ/ระยะที่แน่นอน ต้องเช็คในเกม
> 🛡️ Combatify มี option ให้ปรับเยอะ (เลือกสไตล์ vanilla/combat-test, กลไกโล่หลายแบบ) — เข้าไปจูนได้ใน Mod Menu ตั้งให้ตรงกับเซิร์ฟเพื่อนด้วย

### 👹 MutantMonsters — มินิบอส 5 ตัว
ม็อบกลายพันธุ์ตัวใหญ่ เลือดหนา ตีเจ็บ **เกิดแบบสุ่ม (โอกาสน้อย) ตรงที่ม็อบปกติชนิดนั้นเกิด** — เจอเมื่อไหร่เตรียมรับมือไว้เลย

| มินิบอส | เจอจาก | ดรอป/ของเด่น |
|---|---|---|
| **Mutant Zombie** | โซมบี้ (สุ่ม) | **Hulk Hammer** (ค้อนทุบแรง ๆ) — ตีล้มแล้วยังฟื้นได้ ต้องเอา flint & steel จุดไฟใส่ตอนมันล้มถึงจะตายสนิท |
| **Mutant Skeleton** | สเกเลตัน | ชิ้นส่วนกระดูก (limb/pelvis/rib cage/arms) → เอาไปคราฟต์ **ชุดเกราะ Mutant Skeleton** · **Skull** (เอาไปทำ Chemical X) |
| **Mutant Creeper** | ครีปเปอร์ | **Creeper Shard** (ตัว charged ดรอปเยอะกว่า) เอาไปทำ Creeper Minion |
| **Mutant Enderman** | เอนเดอร์แมน | **Endersoul Hand** (ไอเทมเทเลพอร์ตตัวเอง) + Endersoul Fragment |
| **Spider Pig** | สร้างเอง | ป้อน fermented spider eye ให้หมู แล้วปา **Chemical X** ใส่ |

**ชุดเกราะ Mutant Skeleton** — คราฟต์จากชิ้นส่วนกระดูกที่ดรอปมา ได้ทั้งหมด **4 ชิ้น: boots / leggings / chestplate / arms** (ชุดนี้ **ไม่มี helmet** — อย่างเช่น chestplate = arms + rib cage)

> ⚠️ โบนัสรายชิ้น (กระโดดสูง / วิ่งเร็ว / ขึ้นธนูไว ฯลฯ) **ต้องเช็คในเกม** — ค่าฝังอยู่ในโค้ดมอด หาจากไฟล์ตรง ๆ ไม่ได้

**ของพิเศษอื่น:**
- **Chemical X** — คราฟต์: **obsidian (หรือ crying obsidian)×4 ล้อมของดรอปบอส×1** (Hulk Hammer / Endersoul Hand / Creeper Shard / Mutant Skeleton Skull) → ใช้สร้าง Mutant Snow Golem / Spider Pig
- **Creeper Minion** — ฟักจาก Creeper Minion Egg เป็นสัตว์เลี้ยง ใช้ **Creeper Minion Tracker** ตั้งค่ารัศมีระเบิด/ขี่บ่า/โชว์ชื่อ
- **Hulk Hammer** — ใช้ฆ่า Mutant Zombie ได้ปั๊บ (advancement "Hulk Smash!")

> ⚠️ Mutant Zombie ล้มแล้ว "ไม่ตายจริง" จนกว่าจะจุดไฟทับ — อย่าเพิ่งวางใจ
> 💀 ตัวพวกนี้แรงพอจะฆ่าคนเกราะเหล็กได้ ควรมีโล่ (Combatify) + เกราะดี + อาหารก่อนสู้

### 🔨 Easy Anvils — ทั่งไม่พัง ตีได้ไม่จำกัด
ใช้ทั่ง vanilla ตามปกติ แต่:
- **ทั่งไม่เสื่อม/ไม่แตก** (ไม่มี chipped/damaged anvil) — ทุบใช้ได้ตลอด
- **ตัด "Too Expensive!"** ทิ้ง — รวมเอนแชนต์/ซ่อมได้เรื่อย ๆ ไม่ตันที่ 39 เลเวล
- **เปลี่ยนชื่อใส่สี/ฟอร์แมตได้** — ใช้รหัส `§` (สี, ตัวหนา, เอียง ฯลฯ) ในช่องตั้งชื่อทั่ง

> 💡 คู่กับ Mending: เอาของหลายชิ้นมารวมเอนแชนต์ในทั่งได้ถูกลงเยอะเมื่อไม่มีเพดาน Too Expensive

### 🛏️ Comforts — ถุงนอน & เปลญวน นอนได้ทุกที่
นอนข้ามคืน/ข้ามวันโดย **ไม่ต้องตั้ง spawn** และกัน Phantom ได้ (ตั้งใน config)

| ไอเทม | สูตร | ใช้ยังไง |
|---|---|---|
| **Sleeping Bag** | ขนแกะ×3 เรียงแนวตั้ง (` # `×3) | ย่อตัว (crouch) แล้วคลิกขวาวางพื้น → นอน; **มีโอกาสขาดหลังใช้** |
| **Hammock** (เปลญวน) | ขนแกะ×1 + เชือก×2 + แท่งไม้×2 (` X `/`S#S`/` X `) | ต้องแขวนระหว่าง **Rope and Nail** สองอัน; นอนได้ตอน **กลางวัน** (ข้ามไปกลางคืน) |
| **Rope and Nail** | เชือก×4 + iron ingot×1 (`SS`/`SI`/`S`) | ได้ 2 อัน — สำหรับแขวนเปลญวน |

**ขั้นตอนเปลญวน:** วาง Rope and Nail สองอันห่างกันพอดี → คลิกขวาวาง Hammock เชื่อมระหว่างสองอัน → คลิกขวานอน (กลางวันเท่านั้น)

> 💤 Sleeping Bag = นอนเร็วทุกที่แต่เปลือง (ขาดได้), Hammock = ถาวรกว่าแต่ต้องตั้งเสา + นอนกลางวัน
> 👻 ทั้งคู่ตั้งให้ "พยายามนอน = หยุด Phantom เกิด" ได้แม้ไม่ได้นอนจริง (config: hammocksStopPhantoms / sleepingBagsStopPhantoms)

### ✨ Highlighter — ไฮไลต์ของที่เพิ่งเก็บ
**Client only — ทำงานอัตโนมัติ ไม่มีปุ่ม** ของชิ้นที่เพิ่งเก็บเข้า inventory จะมี **ขอบเรืองไฮไลต์** ในหน้า inventory ให้เห็นว่าได้อะไรมาใหม่ (กันพลาด/กันหาของ)

> หมายเหตุ: เป็นไฮไลต์ "ของในช่อง inventory" ตอนเปิดกระเป๋า ไม่ใช่ไฮไลต์ไอเทมที่ตกอยู่บนพื้นในโลก

## 🔗 คอมโบเด็ด — ใช้คู่กันดี

1. **VeinMiner + เป้ (Pickup/Magnet upgrade)** → ขุดแร่ทั้งสายทีเดียว ของเด้งเข้าเป้เองหมด ไม่ต้องไล่เก็บ
2. **Waystones + เป้** → ตั้งเสาที่เหมือง/ฟาร์ม/บ้าน ขนของเต็มเป้แล้ววาร์ปกลับ ลดเวลาเดินทาง
3. **Comforts/ถุงนอนในเป้ + Waystones** → ออกสำรวจไกล นอนข้ามคืนทุกที่ + กัน Phantom + วาร์ปกลับเมื่อเสร็จ
4. **Combatify (โล่หลายระดับ) + MutantMonsters** → เจอมินิบอส มีโล่ดี ๆ บล็อกไหว รอดสูงขึ้น
5. **Easy Anvils + ของดรอปบอส/Advanced Netherite** → รวมเอนแชนต์เกราะ-อาวุธชุดลุยบอสได้ถูก ไม่ตัน Too Expensive
6. **เกราะ Mutant Skeleton + ธนู** → quick draw + multishot ยิงรัวหลายดอก เปิดศึกระยะไกลกับมินิบอสตัวอื่น
