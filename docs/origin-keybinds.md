# ⌨️ ปุ่มพลัง Active — RPG Origins (เปิดดูปุ๊บกดเป็น)

> ดึงจากไฟล์มอดจริง: `Origins-1.13.0-alpha.12` + `medievalorigins-7.0.4-alpha-1`
> ปุ่มที่ผูกไว้ "จริง" อ่านตรงจาก `options.txt` ของอินสแตนซ์นี้ · คูลดาวน์แปลงจากค่าในไฟล์พลัง (20 ติ๊ก = 1 วิ)

## ⚡ TL;DR
- มอดนี้มี **3 ปุ่มพลังที่ผู้เล่นกดเอง** + ปุ่มดูข้อมูล:
  | ปุ่ม (ตั้งไว้ในเครื่องนี้) | คือ | สถานะ |
  |---|---|---|
  | **`G`** | **Active Power (Primary)** — พลังหลักของเกือบทุกเผ่า | ✅ ผูกไว้แล้ว |
  | *(ยังว่าง)* | **Active Power (Secondary)** — พลังรอง/โหมดสลับหลายเผ่า | ⚠️ **ต้องตั้งเอง** |
  | **`V`** | **Medieval Origins: Tertiary Active** — ปุ่มสาม (Fae/Revenant/Pixie) | ✅ ผูกไว้แล้ว |
  | **`O`** | **View Origin** — เปิดดูข้อมูลเผ่า | ✅ ผูกไว้แล้ว |
  | **`K`** | **Show Powers** — โชว์ทูลทิปพลังที่กดได้ (ตัวช่วยจำ) | ✅ ผูกไว้แล้ว |
- 🔴 **อย่างเดียวที่ต้องตั้งก่อนเล่น = ปุ่ม Secondary** (ไม่ตั้งไว้ = พลังรอง/โหมดสลับของหลายเผ่ากดไม่ออกเลย)
- พลัง active ส่วนใหญ่กองอยู่ที่ปุ่ม **Primary (`G`)** → เผ่าเกือบทั้งหมดแค่กด `G` ก็ปล่อยพลังได้ทันที
- เผ่าที่ "ไม่มีพลัง active เลย" (passive ล้วน) มี **8 เผ่า** — ดูท้ายไฟล์

> ⚠️ ในเครื่องนี้มี **ปุ่มชนกันหลายคู่** (มอดอื่นแย่งปุ่มเดียวกันอยู่) — แวะอ่านหัวข้อ "ปุ่มชนกัน" ก่อนลงเล่นจริง

## 🔧 ตั้งปุ่ม Secondary (ทำครั้งเดียวจบ)

1. เข้าเกม → **Options → Controls → Key Binds**
2. หาแถวชื่อ **"Active Power (Secondary)"** (หมวด Origins) — ตอนนี้จะขึ้นว่า *ไม่ผูก*
3. คลิกที่ช่อง → กดปุ่มที่อยากใช้ (แนะนำ **ปุ่มข้างเมาส์** หรือ `R`* / `C` / `X`)
4. เสร็จ — เผ่าที่ใช้พลัง Secondary (Fae Nourishment, Banshee Spectral, Dwarf/Alfiq/Arachnae โหมดสลับ ฯลฯ) ก็กดได้แล้ว

> 💡 *ตอนนี้ `R` คือ **Combat Roll (กลิ้งหลบ)** — ถ้าจะเอา `R` ไปทำ Secondary ต้องย้าย Combat Roll ไปปุ่มอื่นก่อน เลือกปุ่มว่างมาเลยจะง่ายกว่า
> 💡 อยากเปลี่ยน `G`/`V`/`O` เป็นปุ่มอื่นก็ทำได้ในหน้าเดียวกัน — 3 ตัวนี้แค่ "ผูกไว้ให้ล่วงหน้า" ไม่ใช่ห้ามแก้

## ⚠️ ปุ่มชนกัน (สำคัญ — แก้ก่อนเล่นจริง)

ในอินสแตนซ์นี้มีมอดหลายตัวจองปุ่มเดียวกัน — บางคู่ยิงพร้อมกันด้วย ต้องเข้าไปแก้ที่ **Controls**:

| ปุ่ม | ใครแย่งบ้าง | ผลที่เจอ / ทางแก้ |
|---|---|---|
| **`B`** | Travelers Backpack (เปิดเป้) · JourneyMap (ปักหมุด) · Deeper Darker (boost) | กดทีเดียวทำงานพร้อมกันหลายอย่าง → ย้าย boost/ปักหมุดไปปุ่มอื่น |
| **`V`** | Medieval Tertiary (พลังสาม) · Deeper Darker (transmit) | ถ้าเล่น Fae/Revenant ให้ย้าย transmit ออกไป |
| **`Z`** | JustZoom (ซูม) · Travelers Backpack (สลับเครื่องมือ) | ซูมทีเดียวสลับเครื่องมือไปด้วย → ย้ายอันใดอันหนึ่งออก |
| **`O`** | View Origin · JEI (เปิด/ปิดแผง) | กดดูเผ่าแล้ว JEI หาย → ย้าย JEI ไปปุ่มอื่น (เช่น `\`` ) |

> 💡 จัดปุ่มตาม "เผ่า + คลาส + มอดที่ใช้บ่อย" ให้จบทีเดียว แล้วเล่นยาวได้เลย ไม่ต้องกลับมาแก้อีก

## 🚫 เผ่าที่ไม่มีพลัง active (passive ล้วน — ไม่ต้องกดปุ่ม)

พลังทำงานเองอัตโนมัติ มีแค่บัฟ/จุดอ่อนติดตัว — **8 เผ่า**:

- **Origins พื้นฐาน:** Avian 🟢 · Blazeborn 🔴 · Feline 🟡 · Human ⚪
- **Medieval:** Goblin 🟢 · Moon Elf 🔴 · Plague Victim 🔴 · Wood Elf 🟢

> หมายเหตุ: บางเผ่าในนี้มีพลังที่ "ดูเหมือน active" แต่จริงๆ มันทำงานตามเงื่อนไข ไม่ใช่ปุ่มพลัง เช่น **Moon Elf** ล่องหนเองเมื่อ **ย่อ (Sneak)** ในที่มืด, **Wood Elf** ชาร์จธนูด้วยการ **กดยิงค้าง** — ทั้งคู่ใช้ปุ่มมาตรฐานของเกม ไม่ใช่ปุ่ม Origins
