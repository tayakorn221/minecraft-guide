# ⌨️ ปุ่มพลัง Active — RPG Origins (เปิดดูปุ๊บกดเป็น)

> ดึงจากไฟล์มอดจริง: `Origins-1.13.0-alpha.12` + `medievalorigins-7.0.4-alpha-1`
> ปุ่มที่ผูกไว้ "จริง" อ่านจาก `options.txt` ของอินสแตนซ์นี้ · คูลดาวน์แปลงจากค่าในไฟล์พลัง (20 ติ๊ก = 1 วิ)

## ⚡ TL;DR
- มอดนี้มี **3 ปุ่มพลังที่ผู้เล่นกดเอง** + ปุ่มดูข้อมูล:
  | ปุ่ม (ตั้งไว้ในเครื่องนี้) | คือ | สถานะ |
  |---|---|---|
  | **`G`** | **Active Power (Primary)** — พลังหลักของเกือบทุกเผ่า | ✅ ผูกไว้แล้ว |
  | *(ยังว่าง)* | **Active Power (Secondary)** — พลังรอง/โหมดสลับหลายเผ่า | ⚠️ **ต้องตั้งเอง** |
  | **`V`** | **Medieval Origins: Tertiary Active** — ปุ่มสาม (Fae/Revenant/Pixie) | ✅ ผูกไว้แล้ว |
  | **`O`** | **View Origin** — เปิดดูข้อมูลเผ่า | ✅ ผูกไว้แล้ว |
  | **`K`** | **Show Powers** — โชว์ทูลทิปพลังที่กดได้ (ตัวช่วยจำ) | ✅ ผูกไว้แล้ว |
- 🔴 **สิ่งเดียวที่ต้องตั้งก่อนเล่น = ปุ่ม Secondary** (ไม่ตั้ง = พลังรอง/โหมดสลับของหลายเผ่ากดไม่ได้เลย)
- เผ่าที่พลัง active อยู่ปุ่ม **Primary (`G`)** เยอะสุด → เผ่าส่วนใหญ่แค่กด `G` ก็ใช้พลังได้ทันที
- เผ่าที่ "ไม่มีพลัง active เลย" (passive ล้วน) มี **8 เผ่า** — ดูท้ายไฟล์

> ⚠️ ในเครื่องนี้มี **ปุ่มชนกันหลายคู่** (มอดอื่นแย่งปุ่มเดียวกัน) — ดูหัวข้อ "ปุ่มชนกัน" ก่อนเล่นจริง

---

## 🔧 ตั้งปุ่ม Secondary (ทำครั้งเดียวจบ)

1. เข้าเกม → **Options → Controls → Key Binds**
2. หาแถวชื่อ **"Active Power (Secondary)"** (หมวด Origins) — ตอนนี้ขึ้นว่า *ไม่ผูก*
3. คลิกช่อง → กดปุ่มที่อยากใช้ (แนะนำ **ปุ่มข้างเมาส์** หรือ `R`* / `C` / `X`)
4. เสร็จ — เผ่าที่ใช้พลัง Secondary (Fae Nourishment, Banshee Spectral, Dwarf/Alfiq/Arachnae โหมดสลับ ฯลฯ) กดได้แล้ว

> 💡 *`R` ปัจจุบันคือ **Combat Roll (กลิ้งหลบ)** — ถ้าเอา `R` ไปทำ Secondary ต้องย้าย Combat Roll ไปปุ่มอื่น เลือกปุ่มว่างจะง่ายกว่า
> 💡 อยากเปลี่ยน `G`/`V`/`O` เป็นปุ่มอื่นก็ได้ในหน้าเดียวกัน — 3 ตัวนี้แค่ "ถูกผูกไว้ให้แล้ว" ไม่ใช่ห้ามแก้

## ⚠️ ปุ่มชนกัน (สำคัญ — แก้ก่อนเล่นจริง)

ในอินสแตนซ์นี้มอดหลายตัวจองปุ่มเดียวกัน — บางอันยิงพร้อมกัน ต้องไปแก้ที่ **Controls**:

| ปุ่ม | ใครแย่งบ้าง | ผลที่เจอ / ทางแก้ |
|---|---|---|
| **`B`** | Travelers Backpack (เปิดเป้) · JourneyMap (ปักหมุด) · Deeper Darker (boost) | กดทีเดียวทำงานพร้อมกันหลายอย่าง → ย้าย boost/ปักหมุดไปปุ่มอื่น |
| **`V`** | Medieval Tertiary (พลังสาม) · Deeper Darker (transmit) | ถ้าเล่น Fae/Revenant ให้ย้าย transmit ออก |
| **`Z`** | JustZoom (ซูม) · Travelers Backpack (สลับเครื่องมือ) | ซูมแล้วสลับเครื่องมือพร้อมกัน → ย้ายอันใดอันหนึ่ง |
| **`O`** | View Origin · JEI (เปิด/ปิดแผง) | กดดูเผ่าแล้ว JEI หาย → ย้าย JEI ไปปุ่มอื่น (เช่น `\`` ) |

> 💡 จัดปุ่มตาม "เผ่า + คลาส + มอดที่ใช้บ่อย" ครั้งเดียว แล้วเล่นยาวได้ ไม่ต้องแก้อีก

## 🚫 เผ่าที่ไม่มีพลัง active (passive ล้วน — ไม่ต้องกดปุ่ม)

พลังทำงานเองอัตโนมัติ มีแค่บัฟ/จุดอ่อนติดตัว — **8 เผ่า**:

- **Origins พื้นฐาน:** Avian 🟢 · Blazeborn 🔴 · Feline 🟡 · Human ⚪
- **Medieval:** Goblin 🟢 · Moon Elf 🔴 · Plague Victim 🔴 · Wood Elf 🟢

> หมายเหตุ: บางเผ่าในนี้มีพลังที่ "ดูเหมือน active" แต่จริง ๆ ทำงานตามเงื่อนไข ไม่ใช่ปุ่มพลัง เช่น **Moon Elf** ล่องหนด้วยปุ่ม **ย่อ (Sneak)** ในที่มืด, **Wood Elf** ชาร์จธนูด้วยการ **กดยิงค้าง** — ใช้ปุ่มมาตรฐานของเกม ไม่ใช่ปุ่ม Origins
