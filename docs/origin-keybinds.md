## ⚡ ระบบพลังเผ่า — G / V / I

RPG Origins ให้กดพลังเองผ่าน **3 ปุ่ม** + ปุ่มดูข้อมูล:

| ปุ่ม | คือ | ใช้กับเผ่า |
|---|---|---|
| **`G`** Primary | พลังหลัก | เกือบทุกเผ่าที่มีพลัง active |
| **`V`** Secondary | พลังรอง / โหมดสลับ | Fae · Banshee · Dwarf · Siren · Incubus · High Elf · Arachnae · Alfiq · Revenant |
| **`I`** Tertiary | พลังที่สาม (Medieval) | Fae · Revenant |
| **`O`** / **`K`** | เปิดหน้าเผ่า / โชว์ทูลทิปพลัง | ทุกเผ่า (ตัวช่วยจำ) |

> ✅ ทั้ง 3 ปุ่ม **ผูกไว้ครบแล้ว** — เดิม Secondary เคยว่าง (พลังรองกดไม่ออกเลย) ตอนนี้อยู่ `V` เรียบร้อย ไม่ต้องตั้งเองอีก

## ✅ เรื่องปุ่มชนกัน — จัดเรียบร้อยแล้ว

> ✅ อินสแตนซ์นี้ **จัดปุ่มใหม่ยึดผู้เล่นแล้ว (2 ก.ค. 2026)** — ทุกปุ่มที่กดในโลกทำสิ่งเดียว ไม่ยิงซ้อนกันอีก

ก่อนหน้านี้ปุ่มดีๆ ถูกมอดหลายตัวกองทับ ตอนนี้แก้หมดแล้ว:

| ปุ่ม | เดิมชนกับ | ตอนนี้ |
|---|---|---|
| **`V`** | Tertiary + DeeperDarker + voice + quiver + souls | = **พลังรอง** เดี่ยวๆ |
| **`/`** | sneak + swap + zoom + command + souls | = **command** เดี่ยวๆ (ย่อตัวไป `Shift`, ซูมไป `T`) |
| **`M` / `H` / `F`** | โดน voice/souls แย่ง | = minimap / oracle / กลิ้งหลบ สะอาด |

> 💡 **เปิด Options → Controls แล้วยังเห็นบางปุ่มขึ้นสีแดง/ส้ม — ปกติ ไม่ต้องแก้** เพราะเป็นปุ่มซ้ำ "คนละจอ" (เมนู JEI / หน้าแผนที่เต็มจอ / โหมด creative) เล่นจริงในโลกไม่ชนกันเลย

## 🚫 เผ่าที่ไม่มีพลัง active (passive ล้วน — ไม่ต้องกดปุ่ม)

พลังทำงานเองอัตโนมัติ มีแค่บัฟ/จุดอ่อนติดตัว — **8 เผ่า**:

- **Origins พื้นฐาน:** Avian 🟢 · Blazeborn 🔴 · Feline 🟡 · Human ⚪
- **Medieval:** Goblin 🟢 · Moon Elf 🔴 · Plague Victim 🔴 · Wood Elf 🟢

> หมายเหตุ: บางเผ่ามีพลังที่ "ดูเหมือน active" แต่จริงๆ ทำงานตามเงื่อนไข ไม่ใช่ปุ่มพลัง เช่น **Moon Elf** ล่องหนเองเมื่อ **ย่อ (Sneak)** ในที่มืด, **Wood Elf** ชาร์จธนูด้วยการ **กดยิงค้าง** — ทั้งคู่ใช้ปุ่มมาตรฐานของเกม ไม่ใช่ปุ่ม Origins
