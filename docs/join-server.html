เข้าเซิร์ฟเพื่อน — Fabric 26.1.2 (คู่มือมือใหม่ อ่านจบเข้าได้เลย)

สำหรับเพื่อนที่ ไม่เคยลงมอดมาก่อน — ทำตามทีละข้อ ไม่ต้องรู้อะไรล่วงหน้า

TL;DR (อ่าน 30 วิ)

ต้องมีครบ 3 อย่าง ถึงจะเข้าได้:

  1. Minecraft Java แท้ (premium) เวอร์ชัน 26.1.2 เป๊ะ — ไม่ใช่ Bedrock/มือถือ
  2. Fabric Loader 0.19.2 ติดตั้งทับลงไป (ตัวรันมอด)
  3. ชุดมอดเพื่อน จาก GitHub (19 ตัว) วางในโฟลเดอร์ mods

แล้วเข้าเกม → Multiplayer → Add Server → ใส่ address: supply-adjustment.gl.joinmc.link → Join

⚠️ กฎข้อเดียวที่ห้ามพลาด: เวอร์ชันเกม + Loader + ชุดมอด ต้อง ตรงกับ host เป๊ะ ผิดตัวเดียว = เด้งออก ดูตาราง Troubleshoot ท้ายหน้า

1. เตรียมตัว — เวอร์ชันต้องตรงเป๊ะ

สิ่งที่ต้องมี เวอร์ชัน หมายเหตุ
Minecraft Java Edition 26.1.2 ต้องเป็นบัญชี premium (ซื้อแท้) เท่านั้น
Fabric Loader 0.19.2 ตัวโหลดมอด — ลงทับ Minecraft ปกติ
Java มากับ launcher ปล่อยให้ launcher จัดการ ไม่ต้องลงเอง

💡 ยังไม่มี 26.1.2 ในเครื่อง? เปิด Minecraft Launcher → กดเล่น 26.1.2 หนึ่งครั้งให้ดาวน์โหลดไฟล์เกมก่อน แล้วค่อยลง Fabric

2. ลงทีละสเต็ป

ทางลัด (ถ้ามี one-liner จาก repo)

repo เพื่อนมีสคริปต์ลงให้อัตโนมัติ (โหลด Fabric + ก๊อปมอดให้ครบในคำสั่งเดียว):

  • Windows → install.ps1
  • macOS → install.sh

⚠️ คำสั่งเป๊ะๆ (URL ที่ต้อง copy ไปวาง) อยู่ใน README ของ repo — เปิดดูที่ https://github.com/tayakorn221/mc-26.1.2-friends-pack วิธีรันสั้นๆ: Windows เปิด PowerShell แล้ววางคำสั่งจาก README · macOS เปิด Terminal แล้ววาง

ไม่อยากใช้สคริปต์ก็ทำมือตามด้านล่างได้ ผลเหมือนกัน

ทำมือ — Windows

  1. โหลด Fabric Installer จาก https://fabricmc.net/use → ได้ไฟล์ .exe กดเปิด
  2. แท็บ Client: - Minecraft Version: เลือก 26.1.2 - Loader Version: เลือก 0.19.2 - ติ๊ก Create profile → กด Install
  3. เปิด File Explorer พิมพ์ที่ช่อง address: %appdata%\.minecraft → Enter
  4. ถ้ายังไม่มีโฟลเดอร์ mods ให้สร้างขึ้นมา
  5. เอาไฟล์มอด (ดูข้อ 3) ทั้งหมด วางในโฟลเดอร์ mods
  6. เปิด Minecraft Launcher → เลือกโปรไฟล์ fabric-loader-26.1.2 → Play

ทำมือ — macOS

  1. โหลด Fabric Installer (.jar) จาก https://fabricmc.net/use → ดับเบิลคลิกเปิด (เปิดไม่ได้: คลิกขวา → Open → Open เพื่อข้าม Gatekeeper)
  2. แท็บ Client: Minecraft 26.1.2 + Loader 0.19.2 → Install
  3. Finder → เมนู Go → Go to Folder… → วาง ~/Library/Application Support/minecraft
  4. สร้างโฟลเดอร์ mods ถ้ายังไม่มี → เอาไฟล์มอดทั้งหมดวางในนั้น
  5. เปิด Launcher → โปรไฟล์ fabric-loader-26.1.2 → Play

3. เอามอดมาจากไหน

โหลดจาก GitHub ของ host (public ใครก็โหลดได้):

  • Repo: https://github.com/tayakorn221/mc-26.1.2-friends-pack
  • ไฟล์: ไปที่แท็บ Releases → v1.0.0 → friends-mods.zip (≈ 23 MB)
  • ในไฟล์มี: 19 มอด + readme — แตก zip แล้วเอา .jar ทั้งหมดวางในโฟลเดอร์ mods

มอด 19 ตัวในแพ็ก (ทุกตัว client + ต้องตรงกับ server):

กลุ่ม มอด
ลื่น/ภาพ fabric-api, sodium, lithium, iris, ferritecore
เมนู/HUD modmenu, reese's-sodium-options, mouse-tweaks, jade, appleskin
แผนที่ xaero's-minimap, xaero's-world-map
เกมเพลย์ fallingtree, rrv (ดูสูตรคราฟต์), farmers-delight
โลก/เนื้อหา friends-and-foes, terralith + deps: resourceful-lib, lithostitched

⚠️ อย่าผสมมอดเอง ห้ามเพิ่ม/ลบตัวไหน นอกจาก host บอก — มอดในแพ็กต้อง ครบและตรงชุด กับ server

4. เข้าเซิร์ฟ

  1. host ต้อง เปิด server ค้างไว้ก่อน (เซิร์ฟไม่เปิด = เข้าไม่ได้ทุกกรณี)
  2. ในเกม: Multiplayer → Add Server
  3. ช่อง Server Address วาง:

supply-adjustment.gl.joinmc.link

ไม่ต้องใส่ :25565 ต่อท้าย — address นี้ชี้พอร์ตให้แล้ว 4. Done → ดับเบิลคลิกชื่อเซิร์ฟ → Join

💡 address ตัวนี้เป็น playit.gg free tier อาจเปลี่ยนได้ถ้า host รีสตาร์ต tunnel — ถ้าเข้าไม่ได้ ถาม host ขอ address ล่าสุดก่อนเสมอ

5. Troubleshoot — เข้าไม่ติด (อ่านตารางนี้)

อาการที่เห็น สาเหตุจริง วิธีแก้
เด้งตอน join: "…client/server… mods" หรือชื่อมอด + Missing/Mismatch มอดไม่ครบ / ไม่ตรงชุด กับ server โหลด friends-mods.zip ใหม่ ลบมอดเก่าใน mods ทิ้งให้หมด วางชุดใหม่ครบ 19 ตัว
"Outdated client/server" หรือเวอร์ชันไม่ตรง เกมหรือ Loader ผิดเวอร์ชัน เช็กว่าเล่นโปรไฟล์ fabric-loader-26.1.2 จริง (ไม่ใช่ vanilla/เวอร์ชันอื่น) + Loader = 0.19.2
เซิร์ฟขึ้นเทา / Connection refused / timed out server ปิดอยู่ หรือ address เปลี่ยน บอก host เปิด server + ขอ address playit ล่าสุด แล้วลบเซิร์ฟเก่าใน list สร้างใหม่
Failed to authenticate / Invalid session บัญชีไม่ใช่ premium หรือ session ค้าง ต้องเป็น MC Java แท้ · ออกแล้ว login launcher ใหม่
เกมเปิดไม่ขึ้น/crash ก่อนถึงเมนู มอดไม่ครบ dep หรือมีไฟล์แปลกปน ใน mods ต้องมีเฉพาะ .jar จากแพ็ก — ลบไฟล์อื่นทิ้ง โหลดแพ็กใหม่
เห็นแต่ Bedrock / มือถือเข้าไม่ได้ คนละเกม เซิร์ฟนี้ Java Edition เท่านั้น Bedrock เข้าไม่ได้

⚠️ จุดที่พังบ่อยสุด — เวอร์ชัน/มอดต้องตรงกับ host เป๊ะ

Fabric เช็ก mod ตอน join แบบ ตัวต่อตัว ผิดแม้แต่ตัวเดียวก็เด้ง จำ 3 ข้อนี้:

  1. เวอร์ชันเกม 26.1.2 เป๊ะ — 26.1.1 หรือ 26.1.3 ก็เข้าไม่ได้
  2. Loader 0.19.2 — ดูเลขตอนลง Fabric อย่ากดเวอร์ชันอื่น
  3. ชุดมอด = ชุดเดียวกับ host — ครบ ไม่ขาด ไม่เกิน

🔴 เรื่องที่ host ต้องรู้ (สำคัญมาก): ตอนนี้ server ถูกเพิ่ม ชุดมอดบอส/QoL ชุดใหม่ (2026) เข้าไปแล้ว (เช่น Mutant Monsters, Combatify, Waystones, Traveler's Backpack, Advanced Netherite ฯลฯ) — แต่ชุดนี้ ยังไม่ได้ใส่ใน friends-mods.zip

มอดพวกนี้หลายตัวเป็นแบบ client ต้องมีด้วย ถ้า server รันอยู่แต่เพื่อนมีแค่ 19 ตัวเดิม เพื่อนจะโดนเด้งเพราะมอดไม่ตรง

ทางแก้ (host เลือกอย่างใดอย่างหนึ่ง): - ยังไม่อัปเดตแพ็ก → ปิด/ไม่รันมอดชุดใหม่บน server ชั่วคราว ให้เพื่อนเข้าด้วย 19 ตัวเดิมไปก่อน - อยากใช้ชุดใหม่ → อัปเดต friends-mods.zip + ออก release ใหม่ ส่งให้เพื่อนลงเพิ่ม ทุกคน ก่อนเข้า

(มอด client-only ล้วน เช่น Inventory Profiles Next / FastTrading / Highlighter ไม่กระทบการ join — ไม่ต้องตรงกับ server)

เช็คลิสต์ก่อนกด Join (กลับมาดูเร็ว)

  • เล่น Minecraft Java premium (ไม่ใช่ Bedrock)
  • โปรไฟล์ที่เลือก = fabric-loader-26.1.2 (Loader 0.19.2)
  • โฟลเดอร์ mods มี .jar ครบ 19 ตัว จาก friends-mods.zip — ไม่มีไฟล์อื่นปน
  • host ยืนยันว่า server เปิดอยู่
  • address = supply-adjustment.gl.joinmc.link (ถามว่าเปลี่ยนรึยัง)
  • ถ้า host เปิดมอดชุดใหม่ 2026 → ได้มอดเพิ่มจาก host ครบแล้ว

เข้าไม่ได้หลังเช็กครบ → ส่ง ข้อความ error เต็มๆ ให้ host เทียบกับตาราง Troubleshoot ข้างบน
